# 🧴 Derma Vision – Website

Derma Vision is a simple website for showcasing a skin disease detection project. It provides an interface to upload images and view model predictions.

## 🚀 Features
- Image upload interface  
- Displays prediction results  
- Clean and responsive UI  

## 🧠 Note
The machine learning model is not included here — this repository only contains the website.
